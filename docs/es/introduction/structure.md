# Estructura de Carpetas

La estructura de carpetas del proyecto FandomHub API está organizada para separar los diferentes componentes y funcionalidades. Aquí tienes una breve descripción de la estructura:

```bash
project_fandomhub_api/
├── config/
│   │   ├── management/
│   │   │   └── commands/
│   │   └── settings/
├── apps/
│   ├── animes/
│   ├── characters/
│   ├── clubs/
│   ├── genres/
│   ├── home/
│   ├── mangas/
│   ├── news/
│   ├── persons/
│   ├── playlists/
│   ├── producers/
│   ├── profiles/
│   ├── randoms/
│   ├── recommendations/
│   ├── reviews/
│   ├── tops/
│   ├── users/
│   └── utils/
├── .github/
│   │   ├── ISSUE_TEMPLATE/
│   │   └── workflows/
├── logs/
├── media/
├── nginx/
├── requeriments/
├── scripts/
├── static/
├── templates/
├── manage.py
├── README.md
├── docker-compose.dev.yml
├── .gitignore
```
